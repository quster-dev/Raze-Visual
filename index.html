🛠 Инструкция по установке

Следуйте шагам для правильной работы Raze Visual

Шаг 1: Подготовка Java Для работы актуальных версий Minecraft и мода Raze Visual требуется Java 17.

Убедитесь, что в настройках вашего лаунчера (TLauncher, Prism, MultiMC и др.) выбрана именно Java 17.

Шаг 2: Установка Fabric API Raze Visual работает на базе Fabric, поэтому наличие вспомогательного мода API обязательно.
  • Скачайте актуальную версию Fabric API для 1.16.5.
  • Переместите .jar файл в папку mods.
Путь: %appdata%\.minecraft\mods
Шаг 3: Установка основных папок Найдите папки Raze Visual и Raze VisualHelper.

Скопируйте обе эти папки целиком и вставьте их в ту же директорию mods.

Шаг 4: Запуск игры Откройте лаунчер, выберите версию 1.16.5 Fabric и нажмите «Играть».
